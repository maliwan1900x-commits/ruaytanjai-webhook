🚀 เริ่มต้นใช้งาน
  1. เปิด LINE Developers Console → สร้าง Messaging API channel
  2. คัดลอก Channel Access Token → ใส่ในหน้า "ตั้งค่า"
  3. คัดลอก Webhook URL จากหน้า "Webhook" → ใส่ใน LINE Console
  4. สมัคร Thunder API ที่ thunder.in.th → ใส่ API Key ในหน้า "ตั้งค่า"
  5. เปิด "Use webhook" ใน LINE Developers Console
  6. ส่งรูปสลิปเข้า LINE OA เพื่อทดสอบ
📡 API Endpoints
GET   /api/stats — ดึงสรุปข้อมูล
GET   /api/webhook — ดึงข้อมูลสำหรับ Dashboard
POST   /api/webhook — รับ events จาก LINE
POST   /api/send — ส่งข้อความไปยังลูกค้า
POST   /api/kv?action=... — บันทึก contacts/config
DELETE   /api/stats — ล้างข้อมูลสลิป+events
🔐 Environment Variables
ตัวแปรสถานะคำอธิบาย
KV_REST_API_URLจำเป็นUpstash Redis URL
KV_REST_API_TOKENจำเป็นUpstash Redis Token
LINE_TOKENจำเป็นLINE Channel Access Token
THUNDER_KEYจำเป็นThunder API Key
LINE_CHANNEL_SECRETแนะนำVerify webhook signature
ADMIN_API_KEYแนะนำรหัสเข้า Dashboard
ALLOWED_ORIGINไม่จำเป็นCORS origin (default: *)
💡 เคล็ดลับ
• สลิปกรุงเทพต้องรอ 5-10 นาทีก่อนตรวจได้

• ข้อมูลเก็บถาวรใน Upstash Redis — ไม่หายแม้ Vercel restart
• ใช้ "ค้นหารายชื่อที่หายไป" ใน Webhook tab ถ้าสงสัยว่า userId หาย
